# Cosplay Rental PH

A web platform for renting cosplay costumes in the Philippines. Owners list their costumes, renters browse and book.

## Features
- User / Owner / Admin roles with Firebase Authentication
- Real‑time costume listings and rentals (Firestore)
- Square JPG image upload for costumes
- Email verification and document upload for owners
- PayMongo payment simulation
- Responsive design (mobile/tablet/desktop)
- Light/dark mode

## Setup
1. Clone the repository
2. Create a Firebase project and enable Email/Password Auth
3. Replace `firebase-config.js` with your own config
4. Deploy to Firebase Hosting or open locally
